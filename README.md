### ✅ 커밋 메시지 컨벤션 (규칙)
* `feat:` 새로운 기능 추가 (예: `feat: 계산기 덧셈 로직 구현`)
* `fix:` 버그 수정
* `docs:` 문서 수정 (README 등)
* `refactor:` 코드 리팩토링 (기능은 그대로, 코드만 예쁘게)


### 🌱 잔디(Contribution) 관리 꿀팁
* **양보다 질:** 매일 한 줄이라도 올리는 게 중요하지만, 의미 없는 커밋보다는 그날 배운 핵심 로직 하나를 제대로 올리는 습관을 들이세요.
* **이슈 활용:** 혼자 공부하더라도 "오늘 할 일"을 **Issues**에 등록하고, 완료 후 커밋할 때 `#1`(이슈 번호)을 적으면 자동으로 연결되어 훨씬 전문적으로 보입니다.
